# ClawWork Issues

Public issue tracker and problem-reporting inbox for ClawWork.

Use this repository to report bugs, confusing behavior, broken tools, API/UI errors, and feature requests for ClawWork.

## Before you open an issue

Do **not** include:

- Secrets or API keys
- Full access tokens
- Private customer data
- Sensitive logs
- Credentials, cookies, or authorization headers

If sensitive context is required, open a sanitized issue and write: "sensitive details available for maintainer follow-up."

## Good bug reports include

- What happened
- What you expected to happen
- Steps to reproduce
- Error text, status codes, request paths, screenshots, or timestamps when available
- Impact and any workaround

## Where source-code issues live

The main ClawWork source repository is private. This public repository is the reporting inbox. Maintainers may link, mirror, or escalate issues into private internal tracking when source-level work is needed.
